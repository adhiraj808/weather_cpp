Place weather audio files in this folder.

Supported formats:
- .mp3
- .wav

Use these filenames so the program can find them automatically:
- clear.mp3 or clear.wav
- cloudy.mp3 or cloudy.wav
- rain.mp3 or rain.wav
- storm.mp3 or storm.wav
- mist.mp3 or mist.wav
- snow.mp3 or snow.wav
- wind.mp3 or wind.wav
